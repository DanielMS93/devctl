Create `pkg/tui/panels/patches.go`:

1. `PatchPanel` struct (follows TaskGraphPanel pattern):
   - `width, height int`
   - `focused bool`
   - `patches []tuimsg.AgentPatch`
   - `cursor int`
   - `showDiff bool` — when true, show the full diff of selected patch
   - `vp viewport.Model` — for scrolling diff content

2. `NewPatchPanel() PatchPanel`.

3. `SetPatches(ps tuimsg.PatchSnapshot)` — updates patch list, clamps cursor.

4. `SetSize(w, h int)`, `SetFocused(b bool)`.

5. Navigation: `MoveUp()`, `MoveDown()` — move cursor in patch list.

6. `ToggleDiff()` — toggle between list view and diff view of selected patch. When entering diff view, set viewport content to selected patch's PatchData.

7. `SelectedPatch() *tuimsg.AgentPatch` — returns patch at cursor.

8. `View() string`:
   - If showDiff: render viewport with patch diff, header showing patch title + status, help "Esc=back  a=approve  x=reject"
   - If !showDiff: render patch list:
     - Header: "Agent Patches" + count of draft patches
     - Each row: `[status] title  branch  age`
     - Status badges: draft=yellow, applied=green, approved=cyan
     - Cursor selection highlighting (same pattern as DetailPanel)
     - Help: "Enter=view diff  p=close  a=approve  x=reject"

Wire into root.go:
1. Add `patchPanel panels.PatchPanel` and `showPatches bool` to RootModel
2. In NewRootModel: initialize patchPanel
3. In Update StateEvent handler: `m.patchPanel.SetPatches(msg.Snapshot.Patches)`
4. Add 'p' key binding: toggle showPatches (same pattern as 't' for tasks)
5. When showPatches is true:
   - up/down: navigate patches or scroll diff
   - Enter: toggle diff view
   - 'a' key on a selected patch: show the patch ID and instruct user to run `devctl agent apply <id>` via logBar status
   - 'x' key: similarly show reject instruction
   - Esc: if in diff view, go back to list. If in list, close patch panel.
6. In View(): check showPatches and render patchPanel in place of right panel (same overlay pattern)
7. In propagateSizes: pass dimensions to patchPanel

Update logs.go: add 'p=patches' to the hint bar text.
