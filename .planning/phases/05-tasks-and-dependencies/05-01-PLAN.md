- `tasks` table: id TEXT PRIMARY KEY, description TEXT NOT NULL, state TEXT NOT NULL DEFAULT 'queued' (valid values: queued/running/completed -- "blocked" is computed, never stored), branch TEXT DEFAULT '' (optional linked git branch), worktree_id TEXT DEFAULT '' REFERENCES worktrees(id) ON DELETE SET DEFAULT, repo_id TEXT DEFAULT '' REFERENCES repos(id) ON DELETE CASCADE, created_at INTEGER NOT NULL, updated_at INTEGER NOT NULL

- DROP TABLE IF EXISTS task_deps; DROP TABLE IF EXISTS tasks;

Use CREATE TABLE IF NOT EXISTS and CREATE INDEX IF NOT EXISTS consistent with prior migrations.

```go
package task

type Task struct {
    ID          string `db:"id"`
    Description string `db:"description"`
    State       string `db:"state"`        // queued, running, completed
    Branch      string `db:"branch"`       // optional linked git branch
    WorktreeID  string `db:"worktree_id"`  // optional linked worktree
    RepoID      string `db:"repo_id"`      // scoped to repo
    CreatedAt   int64  `db:"created_at"`
    UpdatedAt   int64  `db:"updated_at"`
}

type TaskStore struct { db *sqlx.DB }
func NewStore(db *sqlx.DB) *TaskStore
```

Methods on TaskStore:
- `Create(ctx, description, repoID string) (Task, error)` -- generates UUID, sets state="queued", timestamps
- `List(ctx context.Context) ([]Task, error)` -- all tasks ordered by created_at DESC
- `ListByRepo(ctx, repoID string) ([]Task, error)` -- tasks for a specific repo
- `Get(ctx, id string) (Task, error)` -- single task by ID (also support prefix match: WHERE id LIKE ? with id+'%', return error if ambiguous)
- `Update(ctx, id string, state string, branch string) error` -- updates state/branch/updated_at. Validate state is one of queued/running/completed (reject "blocked").
- `Delete(ctx, id string) error` -- DELETE FROM tasks WHERE id = ?

Create internal/dependency/store.go:

```go
package dependency

type Dep struct {
    TaskID      string `db:"task_id"`
    DependsOnID string `db:"depends_on_id"`
    CreatedAt   int64  `db:"created_at"`
}

type DepStore struct { db *sqlx.DB }
func NewStore(db *sqlx.DB) *DepStore
```

Methods on DepStore:
- `Add(ctx, taskID, dependsOnID string) error` -- INSERT into task_deps with timestamp. Let DB CHECK constraint catch self-deps.
- `Remove(ctx, taskID, dependsOnID string) error` -- DELETE FROM task_deps WHERE task_id=? AND depends_on_id=?
- `List(ctx, taskID string) ([]Dep, error)` -- deps for one task (what it depends on)
- `ListAll(ctx context.Context) ([]Dep, error)` -- all dependency links (for resolver)

Use github.com/google/uuid for ID generation. Use time.Now().Unix() for timestamps. Import sqlx from github.com/jmoiron/sqlx.
